D4 > GPIO4 / D2 / SDA

D5/D13 > GPIO14 / D5 /SCK

D6/D12 > GPIO12 / D6 / MISO

D7/D11 > GPIO13 / D7 / MOSI

D8 > GPIO0 / D3 - pin upload harus dicabut

D9 > GPIO2 BUILT IN LED / D4

D10 > GPIO15 / SS / D8

A0 > A0

SDA > GPIO4 / D4

BUZZER > D5
